### 总结：
- **`and`**：所有条件都满足时返回 `True`。
- **`or`**：任意一个条件满足时返回 `True`。
- **`not`**：取反操作，将 `True` 转为 `False`，或将 `False` 转为 `True`。
- 逻辑运算符常用于复杂条件的判断和控制流程。
- ==逻辑运算符的优先级：not > and > or==
```python
## 三、逻辑运算符示例（and / or / not）##  
# 患者是否患有慢病  
has_diabetes = True  
has_hypertension = False  
# 是否为高风险人群？（糖尿病 + Hypertension同时存在）  
is_high_risk = has_diabetes and has_hypertension  
print("是否为高风险人群？", is_high_risk)  # False  
# 是否需要筛查？（任意一种疾病）  
needs_screening = has_diabetes or has_hypertension  
print("是否需要筛查？", needs_screening)  # True  
# 是否完全健康？  
is_healthy = not (has_diabetes or has_hypertension)  
print("是否完全健康？", is_healthy)  # False
```
1. **`has_diabetes = True` 和 [has_hypertension = False](vscode-file://vscode-app/Applications/Visual%20Studio%20Code.app/Contents/Resources/app/out/vs/code/electron-sandbox/workbench/workbench.html)**：
    
    - 定义两个布尔变量，分别表示患者是否患有糖尿病和高血压。
2. **`is_high_risk = has_diabetes and has_hypertension`**：
    
    - 使用逻辑运算符 `and` 判断是否为高风险人群。
    - 条件：同时患有糖尿病和高血压。
    - 返回 `False`，因为 [has_hypertension](vscode-file://vscode-app/Applications/Visual%20Studio%20Code.app/Contents/Resources/app/out/vs/code/electron-sandbox/workbench/workbench.html) 是 `False`，两个条件不同时满足。
3. **`needs_screening = has_diabetes or has_hypertension`**：
    
    - 使用逻辑运算符 `or` 判断是否需要筛查。
    - 条件：患有任意一种疾病（糖尿病或高血压）。
    - 返回 `True`，因为 [has_diabetes](vscode-file://vscode-app/Applications/Visual%20Studio%20Code.app/Contents/Resources/app/out/vs/code/electron-sandbox/workbench/workbench.html) 是 `True`，满足至少一个条件。
4. **`is_healthy = not (has_diabetes or has_hypertension)`**：
    
    - 使用逻辑运算符 `not` 判断是否完全健康。
    - 条件：既没有糖尿病，也没有高血压。
    - 返回 `False`，因为 [has_diabetes](vscode-file://vscode-app/Applications/Visual%20Studio%20Code.app/Contents/Resources/app/out/vs/code/electron-sandbox/workbench/workbench.html) 是 `True`，患者不完全健康。
